# DFS State Plugin

Checks for MS Windows(r) DFS-R State and return if DFS is:
 - In Normal Operation
 - With some general Error
 - In Auto Recover Operation
 - In Initial Sincronization
 - Is Initializing
 - Is not initialized

The Agent Plugin doesn't need a Domain User to works.
