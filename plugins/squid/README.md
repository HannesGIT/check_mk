# Checks for Squid Proxy

Checks for Squid3 Cache Health:
 - Squid Client Hits/s
 - Squid Client Requests/s
 - Squid CPU Usage
 - Squid DNS response time
 - Squid Server Requests/s

Squid Source: http://www.squid-cache.org
